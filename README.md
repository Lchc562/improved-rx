# improved-rx
This repository is for showcasing the source code of the paper "Improved Deep Learning-Based Rotational-XOR1 Attacks on Simon32/64 and Simeck32/642".

# Repository Structure
This repository contains six folders corresponding to the modules described in the paper:
- **basic_train**: Simon model training
- **simeck_train**: Simeck model training
- **key_recovery**: Key recovery
- **keyaveraging_train**: Keyaveraging training algorithm
- **neural_bit**: Neutral bit search
- **sensetive_bit**: Sensitive key bit search

## Note on Weight Files
Due to GitHub storage and file size limitations, model weight parameters **cannot be uploaded directly** to this repository.
Users must generate the required files using the code provided in this repository and place them into the corresponding directories before running the relevant code.
